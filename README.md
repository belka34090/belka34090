## Projet principal en cours — FormerFill

**FormerFill** est un projet SaaS privé en cours de développement.

Objectif : automatiser le traitement de documents administratifs, juridiques, santé, fiscaux, comptables et privés.

Le produit vise à :

* lire des PDF scannés, vectoriels ou interactifs ;
* détecter les champs, cases, tableaux et zones de saisie ;
* extraire les données utiles depuis des documents sources ;
* mapper ces données vers des formulaires PDF ;
* générer des PDF remplis, interactifs, éditables et exploitables ;
* appliquer un contrôle qualité final avant export.

Stack principale :

`Python` · `FastAPI` · `Azure Document Intelligence` · `OCR` · `PyMuPDF` · `pdfplumber` · `YAML` · `JSON` · `Tests automatisés`

Le repository reste privé car le projet manipule des documents sensibles. Une version publique de démonstration ou un case study pourra être publiée avec données fictives, architecture documentée et captures anonymisées.
